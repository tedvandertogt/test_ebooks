Section with lists

There are two types of lists: ordered (ol) and unordered (ul). I have used these earlier in this book. In tests, I noticed that the number (nor the dot) is not spoken.

So here are some new test examples.

Ordered list with numbers from number 4 till 6.

  1. This is the first line of an ordered list, with number 4.
  2. This is the second line. Number 5.
  3. Number 6 is this, but the figure is maybe not spoken out. So why would this be correct?

What happens if the same list as above is here without an ‘ol’ and the numbers are typed in as normal text:

4. This is the first line of a few lines with the number 4 typed at the start.

5. This is the second line. Number 5.

6. Number 6 is this. This figure is typed so will be spoken out. Is this better than the listed-method?
