Chapter 2

html-tags

There is a difference between styles with a custom name and the styles which names are official defined in HTML.

In the CSS of an epub is defined how a paragraph or word looks by styles. But there are also universal tags for Bold and Italic. Which one to choose is not obvious: neither how this impacts computer narration.

Here follows a small list.

For Bold text there are two html-tags:

  • <b>-tag. Bold text without an explicit meaning.
  • <strong>-tag. Bold text with the intention to stress that.

For italic text there are 3 html-tags:

  • <i>-tag. Italic text, often used for foreign text and names.
  • <em>-tag. Italic text, meaning to emphasise that text
  • <cite>-tag. Citation, typically used for titles of books and films, and often presented in italics too.

Using these tags is by the way not the obvious choice when making books with programs like InDesign.
