About the book

As the title of this books states, this is a test book. This is the first page following the cover with heading (About the book) above. This first text page is tagged, say marked, with a landmark stating: ‘frontmatter’. In the code of the epub this is called epub:type and this will - if correct - show in the navigation panel of the e-reader. For example under the menu: orientation points.

In the frontmatter of this book several topics will be addressed, like biography, the title page, the colophon, a table of contents, a dedication and a motto. From there the ‘real’ book starts and that is tagged as ‘bodymatter’ or ‘Start of Content’.
