Often then some regular body text follows. This is a bit short. Because we want to show some different options for headers. These will differ in size. The tagging is different too: we use h-tags.

Medium header with a h3-tag

That was the first type of sub header. Headers should be short, do not exceed one line and are not closed by a period.

Small header with a h4-tag

That is a second type of sub header. Hierarchy is important in the use of headers. Sometimes the technical specs force different things than the designer wants.

Sub header with a h5-tag

This is the third type. This tag is used very often. However it is a bit unclear if it can be used anywhere. Must it conform a hierarchy, or to put it differently: must this subhead with h5 tag always be preceded by an h4 and h3.
